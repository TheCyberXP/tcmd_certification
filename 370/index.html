Module Distribution

Variety of licensing options


Distributed through Marketplace


Also, GitHub, Blogs, etc.


Variety of distribution methods are used. Modules can be sold or distributed via marketplace.appcelerator.com. Appcelerator-supported modules are available through support.appcelerator.com. Open-source and unofficial modules are often distributed by their authors via GitHub, blogs, and so forth.

Installing Modules

Two install options: for single project or all projects

    Install for single project:
  1. Download zip file
  2. Place in project's root directory
  3. Build once — to unzip and create necessary folders
  4. Add to tiapp.xml and JavaScript
Installing for all projects doesn't add overhead to your projects, so is probably the best option. We'll cover how to update tiapp.xml and JavaScript on an upcoming slide.

Installing Modules, Continued

    Install for all projects:
  1. Download zip file
  2. Unzip to %TITANIUM_INSTALL_DIR%/modules directory
  3. Add to tiapp.xml and JavaScript
Any future projects you develop on this computer would then be able to use the module.

The Titanium install directory varies by platform and operating system version. For example, on OS X Lion, it would be ~/Library/Application Support/Titanium.
