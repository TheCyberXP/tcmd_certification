Lab: Using Alloy Widgets

Mission: In this lab assignment, you will implement an Alloy widget in an app.

Specification: To successfully complete this lab, you must develop an app that meets the following description:

  • Implements the ButtonGrid Alloy widget.
  • Displays five buttons in a grid with custom background colors, icons, and titles as described below.
  • Includes a click event listener on one of the buttons that displays a simple alert message.
1. Download the graphic assets needed for this lab from S3: 315_ui (starting).
2. Create a new Titanium Alloy project, giving it a name and app ID of your choice. Extract, then copy the images you downloaded into your app/assets/images folder.
3. Update the config.json file to require in the com.appcelerator.buttongrid component. As of this writing, version 1.0 is the current version. (See Resources below for a link to this widget's documentation.)
4. Update the index view to add the tag necessary to display the button grid widget in a window.
5. Update the index controller to implement four buttons as follows:
Button titleidbackgroundColor
Notes write 'gray'
Ideas lightbulb (default)
Chat bubble (default)
People person (default)
Pictures pictures (default)
6. Add a click event listener to the Pictures button. Display an alert dialog box when the button is clicked.
7. Also in the init() method for the button grid, define these properties:
  • Button width: 100 (handset) or 200 (tablets)
  • Button height: 100 (handset) or 200 (tablets)
  • backgroundColor: #660000
  • backgroundSelectedColor: #A30000
  • duration: 800 milliseconds
8. Build your app for the simulator/emulator. Click the Pictures button (the alert should be displayed).
9. Finally, download the widget demo (S3: 315_ui_demo) and explore the code that was used to create this widget. Specifically, explore each of the files in the app/widgets directory. Then also look at the index view and controller to see how this widget is implemented in the app.

Summary

In this lab, you implemented an Alloy widget and set various startup parameters.
